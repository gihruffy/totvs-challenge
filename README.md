# TOTVS Challange

Este projeto é uma implementação do Clean Arch com conceitos de DDD utilizando .net core 3.1;

## Getting Started

Para rodar esse projeto, você precisa ter instalado o .net core SDK 3.1 em seu sistema operacional [.Net Core 3.1](https://dotnet.microsoft.com/download/dotnet-core/3.1)


É recomendavel que para alterações você instale um dos seguintes editores: 
[Visual Studio](https://visualstudio.microsoft.com/pt-br/vs/)
[Visual Studio Code](https://code.visualstudio.com/download)

### Rodando a aplicação

Abra o terminal do seu computador e execute os seguintes comandos

```powershell
dotnet build;
```

```powershell
cd TOTVSChallenge.API; 
```

```powershell
dotnet run;
```

## Rodando os testes

Na raiz da aplicação execute os seguintes comandos

```powershell
dotnet test
```
